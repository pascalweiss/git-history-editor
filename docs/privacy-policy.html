Privacy Policy

Last updated: February 2026

Summary

Git History Editor does not collect, store, or transmit any personal data. The app runs entirely on your local machine.

Data Collection

Git History Editor collects no data whatsoever. Specifically:

  • No analytics or telemetry
  • No crash reports
  • No usage tracking
  • No account creation
  • No cookies or local tracking identifiers

Local Data

The app stores a small amount of data locally on your device:

  • Recent repositories list — The file paths of repositories you have recently opened, stored locally so you can quickly reopen them. This data never leaves your device.
  • Window preferences — Window size and position, managed by the operating system.

Git Repository Access

Git History Editor reads and modifies Git repositories that you explicitly select via the file picker. It accesses only the repositories you choose and operates entirely locally using the libgit2 library. No repository data is ever transmitted over the network.

Network Access

The app makes a single outbound network request to check for software updates. This request goes to GitHub (github.com) to determine if a newer version of the app is available. No personal data is included in this request.

Third-Party Services

Git History Editor does not integrate with any third-party services, advertising networks, or analytics platforms.

Children's Privacy

The app does not collect any data from anyone, including children under the age of 13.

Changes to This Policy

If this policy changes, the updated version will be published at this same URL with a new "last updated" date.

Contact

If you have questions about this privacy policy, please open an issue on the project's repository.
